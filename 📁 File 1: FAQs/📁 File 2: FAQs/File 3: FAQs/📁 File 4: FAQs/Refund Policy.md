# Refund Policy

## Overview

Refund requests are accepted within 14 days of purchase.

## Eligibility

- Valid order number
- Purchase meets refund conditions

## Process

1. Submit a refund request.
2. Verification by the support team.
3. Refund approval or rejection.
4. Processing within 5–7 business days.

## Exceptions

- Gift cards
- Used digital products
- Promotional items

## Contact Support

Include your Order ID when requesting a refund.
